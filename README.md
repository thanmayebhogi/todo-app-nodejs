# Todo App 📝

A simple Todo List application built using Node.js, Express.js, MongoDB, Mongoose, and EJS.

## Features
- Add tasks
- Delete individual tasks
- Delete all tasks
- Store tasks in MongoDB

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS

## Run Locally

```bash
npm install
node app.js
```
